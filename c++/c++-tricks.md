# c++ tricks

## array as input 

### example 1, 

```cpp
double getAverage(int arr[], int size)
{
  int    i, sum = 0;
  double avg;
  for (i = 0; i < size; ++i)
  {
    sum += arr[i];
   }
  avg = double(sum) / size;
  return avg;
}
```
